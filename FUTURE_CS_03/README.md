Overview

This repository contains an API Security Assessment Report prepared as part of a cybersecurity internship project. The assessment was conducted against a demo API environment to identify common API security weaknesses.

Key Findings
a. Unauthenticated Access to API Endpoint
b. Excessive Data Exposure
c. Lack of Rate Limiting

Methodology
a. Manual Testing
b. API Request Analysis
c. Response Validation

References
a. OWASP API Security Top 10
b. NIST SP 800-53 Rev. 5

Disclaimer

This report was created for educational and internship purposes using an authorized demo environment.
