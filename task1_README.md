Vulnerability Assessment Report for a Live Website

Overview

This repository contains a Web Application Vulnerability Assessment Report prepared as part of a cybersecurity internship project. The assessment focused on identifying common web security weaknesses.

Key Findings
a. Missing Security Headers
b. Insecure Cookie Attributes
c. Information Disclosure via HTTP Headers

Methodology
a. Manual Testing
b. HTTP Header Analysis
c. Security Configuration Review

References
a. OWASP Web Security Testing Guide
b. OWASP Secure Headers Project
c. NIST SP 800-53 Rev. 5

Disclaimer
This report was created for educational and internship purposes using an authorized demo environment.
